// Week by Week
Full Curriculum ‚Äî 6 Weeks
  • Image representation: pixels, channels, bit depth, aspect ratio
  • Colour spaces: RGB, BGR (OpenCV default), HSV, LAB, grayscale
  • Histogram analysis and equalisation for contrast enhancement
  • Geometric transforms: resize, rotate, flip, perspective warp
  • Drawing and annotation: rectangles, circles, text overlays
  • Thresholding: global, adaptive (Otsu), colour-based segmentation
  • Morphological operations: erosion, dilation, opening, closing
  • Project: document scanner with perspective correction
  • Spatial filtering: Gaussian, median, bilateral ‚Äî noise vs structure tradeoffs
  • Frequency domain: FFT, low-pass/high-pass filters, Fourier analysis
  • Edge detection: Sobel, Laplacian, Canny ‚Äî parameter tuning
  • Contour detection, approximation, shape moments
  • Feature detectors: Harris corners, FAST, SIFT, ORB
  • Feature matching: BFMatcher, FLANN, RANSAC for outlier rejection
  • Homography and image stitching ‚Äî build your own panorama
  • Template matching for object localisation
  • HOG (Histogram of Oriented Gradients) + SVM detector
  • Haar cascades: face, eye, smile detection
  • Background subtraction: MOG2, KNN, scene modelling
  • Optical flow: Lucas-Kanade, Farneback dense flow
  • Object tracking: CSRT, KCF, MOSSE trackers
  • Multi-object tracking: DeepSORT fundamentals
  • Video pipeline: frame capture ‚Üí process ‚Üí annotate ‚Üí display
  • Real-time webcam: motion detection and people counting demo
  • CNN review: conv layers, pooling, BN, dropout ‚Äî fast refresh
  • Transfer learning on domain-specific datasets (medical, satellite, industrial)
  • Data augmentation: Albumentations library for CV pipelines
  • YOLOv8: architecture, fine-tune on custom detection dataset
  • Grad-CAM and Score-CAM: visualise CNN attention for explainability
  • Anomaly detection with PatchCore on MVTec industrial dataset
  • Evaluation: mAP, IoU, confusion matrix for object detection
  • Domain 1 project: manufacturing defect detection pipeline
  • Medical imaging: DICOM format, windowing, pixel spacing
  • Chest X-ray classification with CheXNet-style fine-tuning
  • Retinal fundus image analysis: optic disc detection
  • Medical image segmentation: U-Net on chest X-ray lung masks
  • Satellite imagery: multispectral bands, NDVI vegetation index
  • Land cover classification: EuroSAT dataset with ResNet
  • Visual product search: CLIP embeddings + FAISS for retail
  • Domain 2 project: medical image analysis pipeline
  • Model export: ONNX, TorchScript for framework-agnostic inference
  • FastAPI CV service: accept image upload, return predictions
  • Optimisation: quantisation, pruning, distillation ‚Äî faster inference
  • Edge AI introduction: deploy OpenCV + ONNX on Raspberry Pi
  • Jetson Nano basics: GPU-accelerated inference at the edge
  • Streamlit / Gradio CV demo deployment ‚Äî share your work publicly
  • Capstone: end-to-end CV system targeting a specific industry domain
  • Live demo to batch + mentor review + Gold badge issuance
